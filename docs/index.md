# About the bicycle-led-pcb

This is the documentation for the bicycle LED pcb software project.

The project is about creating a PCB that can control multiple lights and other things on a bicycle, and read out various
sensors. It also has a built-in screen that can show all kinds of information to the bicycle rider.

### Getting Started

You are interested in the project? Great! You can find how to get started in the [Getting Started](getting-started/hardware.md) section.
